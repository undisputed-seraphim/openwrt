# Nuvoton MINI51

This is the series name that includes the chip of interest, MINI54FDE.

https://www.nuvoton.com/products/microcontrollers/arm-cortex-m0-mcus/mini51-base-series/

Nomenclature breakdown:
4 - 4 banks of 4kb flash, 16kb total
F - TSSOP20 package

Board support package (BSP) of the series as a whole can be found here:
https://github.com/OpenNuvoton/Mini51BSP
